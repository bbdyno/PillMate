# 복약메이트(DoseMate) 개인정보 처리방침

**시행일자**: 2025년 12월 11일

복약메이트(이하 "서비스")는 「개인정보 보호법」 제30조에 따라 이용자의 개인정보를 보호하고 이와 관련한 고충을 신속하고 원활하게 처리할 수 있도록 하기 위하여 다음과 같이 개인정보 처리방침을 수립·공개합니다.

## 제1조 (개인정보의 처리 목적)

본 서비스는 **서버에 개인정보를 수집하거나 저장하지 않습니다**. 모든 데이터는 이용자의 기기(iPhone, iPad 등)에 로컬로 저장됩니다.

서비스는 다음의 목적을 위하여 이용자 기기 내에서 개인정보를 처리합니다:

1. **약물 복용 관리**
   - 약물 정보, 복용 일정, 복약 기록 관리
   - 복약 알림 제공

2. **환자(피보호자) 관리**
   - 환자 정보 등록 및 관리
   - 환자별 약물 및 복약 기록 관리

3. **진료 예약 관리**
   - 진료 예약 일정 관리
   - 병원 및 의사 정보 관리

4. **통계 및 분석**
   - 복약 준수율 계산
   - 복약 패턴 분석

5. **HealthKit 연동** (프리미엄, 선택적)
   - 복약 기록을 Apple Health 앱에 저장
   - 건강 데이터 조회 및 분석

6. **iCloud 동기화** (프리미엄, 선택적)
   - 여러 기기 간 데이터 동기화
   - 데이터 백업 및 복원

## 제2조 (처리하는 개인정보의 항목)

본 서비스는 다음의 개인정보를 **이용자 기기 내에서만** 처리합니다:

### 1. 필수 정보
- **약물 정보**: 약물명, 용량, 복용 시간, 복용 빈도, 복용 기간
- **복약 기록**: 복약 일시, 복용 여부, 건너뛰기 사유
- **알림 설정**: 알림 시간, 스누즈 설정, 알림 활성화 여부

### 2. 선택 정보
- **환자 정보**: 이름, 생년월일, 관계, 프로필 색상, 메모
- **보호자 정보**: 이름, 전화번호, 이메일, 관계, 알림 설정
- **진료 예약 정보**: 병원명, 의사명, 진료과, 예약 일시, 메모
- **약물 사진**: 약물 외관 사진
- **재고 정보**: 현재 재고량, 재고 부족 알림 설정
- **건강 지표**: 혈압, 혈당 등 (이용자가 직접 입력하는 경우)

### 3. 자동 수집 정보
- **기기 정보**: iOS 버전 (앱 호환성 확인용, 로컬 저장)
- **앱 사용 기록**: 마지막 접속 시간 (로컬 저장)

### 4. HealthKit 데이터 (선택적, 프리미엄)
- Apple Health 앱에 저장된 복약 기록
- 건강 데이터 읽기/쓰기 권한

### 5. iCloud 동기화 데이터 (선택적, 프리미엄)
- 위 1~4항의 모든 데이터가 Apple iCloud에 암호화되어 저장됨
- Apple iCloud 서비스 약관이 적용됨

**중요**: 위의 모든 정보는 서비스 제공자의 서버로 전송되거나 저장되지 않습니다.

## 제3조 (개인정보의 처리 및 보유 기간)

1. **보유 기간**: 이용자가 앱을 삭제하거나 데이터를 삭제할 때까지
2. **보유 장소**: 이용자의 기기 (iPhone, iPad 등)
3. **iCloud 동기화 사용 시**: Apple iCloud 서버 (Apple 개인정보 처리방침 적용)
4. **데이터 삭제**:
   - 앱 삭제 시 모든 데이터 자동 삭제
   - 설정 > 데이터 > 모든 데이터 삭제 기능 제공
   - iCloud 동기화 데이터는 iCloud 설정에서 별도 삭제 필요

## 제4조 (개인정보의 제3자 제공)

1. 본 서비스는 이용자의 개인정보를 제3자에게 제공하지 않습니다.
2. 다만, 다음의 경우는 예외로 합니다:
   - **보호자 알림**: 이용자가 설정한 보호자에게 복약 미이행 알림 전송 (이용자의 명시적 동의)
   - **Apple Services**: iCloud 동기화 사용 시 Apple Inc.에 데이터 저장 (이용자의 선택적 동의)
   - **법령의 규정**: 법령에 특별한 규정이 있거나 법령상 의무를 준수하기 위한 경우

## 제5조 (개인정보처리의 위탁)

본 서비스는 개인정보 처리업무를 외부에 위탁하지 않습니다.

다만, 다음의 서비스는 이용자의 선택에 따라 사용됩니다:
- **Apple iCloud**: 데이터 동기화 및 백업 (프리미엄, 선택적)
- **Apple HealthKit**: 건강 데이터 연동 (프리미엄, 선택적)
- **Apple Push Notification Service (APNs)**: 알림 전송

위 서비스들은 Apple Inc.가 제공하며, Apple의 개인정보 처리방침이 적용됩니다.

## 제6조 (이용자 및 법정대리인의 권리와 행사 방법)

1. 이용자는 언제든지 다음의 권리를 행사할 수 있습니다:
   - 개인정보 열람 요구: 앱 내에서 직접 확인
   - 개인정보 정정 요구: 앱 내에서 직접 수정
   - 개인정보 삭제 요구: 앱 내 설정에서 삭제 또는 앱 삭제
   - 개인정보 처리정지 요구: 해당 기능 비활성화 또는 앱 사용 중단

2. 만 14세 미만 아동의 경우:
   - 법정대리인의 동의가 필요합니다
   - 법정대리인은 아동의 개인정보를 열람, 정정, 삭제할 수 있습니다

## 제7조 (개인정보의 파기)

1. **파기 시점**:
   - 앱 삭제 시 즉시 파기
   - 이용자가 "모든 데이터 삭제" 실행 시 즉시 파기
   - iCloud 데이터는 iCloud 설정에서 별도 삭제 필요

2. **파기 방법**:
   - 전자적 파일: 기술적 방법을 사용하여 복구 불가능하게 영구 삭제
   - iCloud 데이터: Apple의 데이터 파기 정책 적용

## 제8조 (개인정보의 안전성 확보 조치)

본 서비스는 다음과 같은 안전성 확보 조치를 취하고 있습니다:

1. **기술적 조치**:
   - iOS 샌드박스를 통한 데이터 격리
   - iOS Keychain을 사용한 민감 정보 암호화
   - SwiftData 프레임워크의 보안 기능 활용
   - iCloud 동기화 시 end-to-end 암호화

2. **관리적 조치**:
   - 개인정보 취급자 최소화 (개발자만 앱 코드 접근 가능)
   - 서버에 개인정보를 저장하지 않음으로써 유출 위험 원천 차단

3. **물리적 조치**:
   - 이용자 기기의 잠금 기능 활용 (Face ID, Touch ID, 암호)
   - 기기 분실 시 "나의 iPhone 찾기" 기능 활용 권장

## 제9조 (개인정보 자동 수집 장치의 설치·운영 및 거부)

본 서비스는 쿠키(cookie) 등 개인정보를 자동으로 수집하는 장치를 사용하지 않습니다.

## 제10조 (개인정보 보호책임자)

개인정보 처리에 관한 업무를 총괄해서 책임지고, 개인정보 처리와 관련한 정보주체의 불만처리 및 피해구제 등을 위하여 아래와 같이 개인정보 보호책임자를 지정하고 있습니다.

**개인정보 보호책임자**
- 성명: [개발자 이름]
- 직책: 개발자
- 연락처: support@dosemate.app (예시)

※ 개인정보 보호 담당부서로 연결됩니다.

**개인정보 보호 담당부서**
- 부서명: 개발팀
- 이메일: support@dosemate.app (예시)

## 제11조 (권익침해 구제 방법)

이용자는 개인정보침해로 인한 구제를 받기 위하여 개인정보분쟁조정위원회, 한국인터넷진흥원 개인정보침해신고센터 등에 분쟁해결이나 상담 등을 신청할 수 있습니다.

- **개인정보분쟁조정위원회**: 1833-6972 (www.kopico.go.kr)
- **개인정보침해신고센터**: 118 (privacy.kisa.or.kr)
- **대검찰청 사이버범죄수사단**: 1301 (www.spo.go.kr)
- **경찰청 사이버안전국**: 182 (cyberbureau.police.go.kr)

## 제12조 (개인정보 처리방침의 변경)

1. 이 개인정보 처리방침은 2024년 12월 11일부터 적용됩니다.
2. 개인정보 처리방침이 변경되는 경우 변경 사항을 앱 내 공지사항 또는 개별 통지를 통해 공지합니다.
3. 변경된 개인정보 처리방침은 공지한 날로부터 7일 후부터 효력이 발생합니다.

## 제13조 (보호자 알림 기능 관련)

1. **연락처 정보 처리**:
   - 보호자의 이름, 전화번호, 이메일은 이용자 기기에만 저장됩니다
   - 서비스 제공자는 보호자 정보를 수집하거나 저장하지 않습니다
   - 알림 전송 시 iOS 시스템 기능(메시지, 전화 등)을 통해 직접 전송됩니다

2. **보호자 동의**:
   - 보호자 등록 시 반드시 해당 보호자의 사전 동의를 받아야 합니다
   - 보호자는 언제든지 알림 수신을 거부할 수 있습니다

3. **데이터 전송**:
   - 복약 미이행 알림만 전송되며, 약물 상세 정보는 전송되지 않습니다
   - 보호자에게 전송되는 정보: 환자 이름, 복약 미이행 시간

## 제14조 (HealthKit 데이터 처리)

1. **HealthKit 권한** (프리미엄, 선택적):
   - 복약 기록을 Apple Health 앱에 저장하기 위한 쓰기 권한
   - 건강 데이터를 조회하기 위한 읽기 권한 (선택적)

2. **데이터 처리**:
   - HealthKit 데이터는 Apple Health 앱에만 저장됩니다
   - 서비스 제공자는 HealthKit 데이터를 수집, 저장, 분석하지 않습니다
   - HealthKit 데이터는 Apple의 개인정보 처리방침이 적용됩니다

3. **권한 철회**:
   - iOS 설정 > 건강 > 데이터 접근 및 기기에서 언제든지 권한 철회 가능
   - 권한 철회 시 HealthKit 연동 기능 사용 불가

## 제15조 (iCloud 동기화)

1. **iCloud 동기화** (프리미엄, 선택적):
   - 이용자가 활성화한 경우에만 작동
   - 모든 약물, 환자, 보호자, 진료 예약 데이터가 iCloud에 저장됩니다
   - iCloud 데이터는 Apple에 의해 암호화됩니다

2. **Apple iCloud 정책**:
   - iCloud에 저장되는 데이터는 Apple Inc.의 개인정보 처리방침이 적용됩니다
   - Apple 개인정보 처리방침: https://www.apple.com/kr/privacy/

3. **비활성화**:
   - 앱 설정에서 언제든지 iCloud 동기화 비활성화 가능
   - 비활성화 후 iCloud 데이터는 수동으로 삭제해야 합니다

## 제16조 (데이터 내보내기 및 가져오기)

1. **데이터 내보내기** (프리미엄):
   - JSON 파일 형식으로 모든 데이터 내보내기 가능
   - 내보낸 파일은 이용자가 직접 관리
   - 내보낸 파일에는 모든 개인정보가 포함되므로 안전하게 보관 필요

2. **데이터 가져오기** (프리미엄):
   - 내보낸 JSON 파일을 다시 가져오기 가능
   - 가져온 데이터는 기존 데이터와 병합 또는 대체

## 제17조 (아동의 개인정보 보호)

1. 본 서비스는 만 14세 미만 아동의 개인정보를 수집하지 않습니다.
2. 만 14세 미만 아동이 서비스를 이용하는 경우 법정대리인의 동의가 필요합니다.
3. 법정대리인은 아동의 개인정보에 대한 열람, 정정, 삭제를 요청할 수 있습니다.

**시행일**: 본 방침은 2024년 12월 11일부터 시행됩니다.
