## Key Points / 주요 사항

### English
- DoseMate is a **local-based app** that does not operate a server
- The developer does not collect or transmit user data to external servers
- All data is stored only on your device
- HealthKit integration is **planned for future updates** (not currently available)

### 한국어
- DoseMate는 **서버를 운영하지 않는 로컬 기반 앱**입니다
- 개발자는 사용자 데이터를 수집하거나 외부 서버로 전송하지 않습니다
- 모든 데이터는 사용자의 기기에만 저장됩니다
- HealthKit 연동은 **추후 업데이트 예정**입니다 (현재 제공되지 않음)
